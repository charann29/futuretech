Main Server (localhost:5000):
  GET  /                     - Landing page
  GET  /test                 - FSAT test
  GET  /resume               - Quick resume builder
  GET  /builder/*            - Advanced resume builder (React)

  POST /api/resume/process   - Advanced resume (proxy to Python)
  POST /api/resume/generate  - Generate PDF (proxy to Python)

Python Service (localhost:8000):
  GET  /                     - Health check
  POST /process              - Process & enhance resume
  POST /generate             - Generate PDF
