Complete these 4 steps to go live:

[ ] 1. SETUP SUPABASE DATABASE
   ────────────────────────────
   → Open: https://app.supabase.com/project/gsqpobjgxrkeyxfqlakb/sql
   → Open file: supabase-setup.sql
   → Copy all content
   → Paste in SQL Editor
   → Click "Run"
   → Verify tables created: leads, test_results, resumes

[ ] 2. ENABLE GOOGLE OAUTH
   ────────────────────────
   → Open: https://app.supabase.com/project/gsqpobjgxrkeyxfqlakb/auth/providers
   → Find "Google" provider
   → Click "Enable"
   → Add your Google OAuth credentials:
     - Client ID: [from Google Cloud Console]
     - Client Secret: [from Google Cloud Console]
   → Add Redirect URLs:
     - http://localhost:5000/index.html
     - [your-production-url]/index.html

   Don't have Google OAuth credentials?
   → https://console.cloud.google.com/apis/credentials
   → Create OAuth 2.0 Client ID
   → Application type: Web application
   → Authorized redirect URIs: Add your Supabase callback URL

[ ] 3. ADD ANTHROPIC API KEY
   ─────────────────────────
   → Open file: .env
   → Find line: ANTHROPIC_API_KEY=
   → Add your key: ANTHROPIC_API_KEY=sk-ant-your-key-here
   → Save file

   Don't have Anthropic API key?
   → https://console.anthropic.com/settings/keys
   → Create new key
   → Copy and paste

[ ] 4. START SERVER
   ────────────────
   → Open terminal
   → Run: npm start
   → Server starts on: http://localhost:5000
   → Test Google sign-in
   → Test FSAT test flow
   → Test AI Resume builder
